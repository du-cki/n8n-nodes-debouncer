# n8n-nodes-debounce

Provides an [n8n](https://n8n.io/) node that delays workflow execution until wait milliseconds have passed since its last invocation for a given identifier.

## Installation

Follow the [official n8n documentation](https://docs.n8n.io/integrations/community-nodes/installation/) to install this node in your n8n instance.
